# Phase 0 — Schema Migration & Drizzle Types

**Status:** Complete

---

## Purpose

Land the schema generalization that turns campaigns into kinded
experiments. New enum, new columns, new CHECK constraints, backfill of
the model snapshot for existing campaigns. No handler logic in this
phase.

---

## Scope

**This phase DOES:**

- Adds `campaign_kind` enum.
- Adds `kind`, `pinned_provider_model_id`, `pinned_model_snapshot`,
  `pinned_system_prompt` columns to `campaigns`.
- Adds `kind`, `variant_text` to `campaign_models`; drops NOT NULL on
  `provider_model_id`.
- CHECK constraints enforcing per-kind nullability.
- Backfill `pinned_model_snapshot` for existing model-arena campaigns.
- Updates Drizzle schema + exported types.

**This phase does NOT:**

- Touch any handler logic (Phase 1).
- Touch any UI (Phase 2).
- Wire feature flags (Phase 1).
