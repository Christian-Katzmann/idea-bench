# P1-C: Standalone-Variants Verbatim Wiring

Wire the `standaloneVariants` flag end-to-end so a prompt-arena variant
runs verbatim when the operator opts in, instead of receiving an empty
`{{input}}` substitution.

Closes the `TODO(plan-05/p1-c)` left by P1-A in `AdvancedDisclosure`.
